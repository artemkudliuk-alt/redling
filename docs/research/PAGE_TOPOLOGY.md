# Топология страниц и разделов (Page Topology Blueprint)

Этот документ определяет структуру разметки, иерархию слоев (`z-index`) и порядок следования разделов на новом лендинге **Redling Hotel**.

---

## 1. Иерархия слоев и макет (DOM Structure & Layering)

Проект представляет собой единую страницу с фиксированным сайдбаром слева и скроллящимся контентом справа.

```
+--------------------------------------------------------------------------+
|  [page-side] (Fixed, 285px) |  [main-content] (Margin-left: 285px)       |
|                             |                                            |
|  * Logo                     |  +--------------------------------------+  |
|  * Navigation Menu          |  | [video-background-container] (Fixed)  |  |
|  * Promotion Button         |  |  * <video>                           |  |
|  * Phone & Dropdown         |  |  * Dark overlay                      |  |
|  * Social Links             |  +--------------------------------------+  |
|  * Language Selector        |  | [sections-container] (Scrollable)    |  |
|  * Copyright                |  |  * Section 1: Hero                   |  |
|                             |  |  * Section 2: Rooms                  |  |
|                             |  |  * Section 3: Pool & Summer          |  |
|                             |  |  * Section 4: Restaurant             |  |
|                             |  |  * Section 5: Atmosphere             |  |
|                             |  |  * Section 6: Location & Contacts    |  |
|                             |  +--------------------------------------+  |
+--------------------------------------------------------------------------+
```

### Слои (Z-Index Hierarchy)
* **z-index: 1**: `video-background-container` (фоновое видео).
* **z-index: 2**: `sections-container` (текстовые карточки и секции, которые скроллятся поверх видео).
* **z-index: 10**: `page-side` (сайдбар слева, который всегда виден поверх всех остальных блоков на десктопе).
* **z-index: 20**: Выпадающее меню мессенджеров телефона (должно открываться поверх сайдбара).

## 2. Разделы правой контентной области (Right Content Sections)

Каждая секция занимает минимум `100vh` по высоте для обеспечения достаточного пространства скролла, чтобы пользователь мог рассмотреть видео-анимацию на фоне каждого блока.

### 1. Первый экран (Hero Block) — `#hero-section`
* **Высота**: `100vh`, дисплей `flex`, центрирование содержимого.
* **Текстовое наполнение**:
  * Заголовок (H1): `REDLING HOTEL — ваш уютный дом у моря в Одессе`
  * Подзаголовок (p): `Премиальный комфорт для семейного отдыха и деловых поездок круглый год. Всего 500 метров до пляжа, вдали от городского шума.`
  * Кнопка (CTA): `Забронировать номер`
* **Фоновое видео**: Начальные секунды ролика (красивые общие виды отеля с высоты).

### 2. Номера (Rooms) — `#rooms-section`
* **Высота**: `100vh`, позиционирование карточки слева или по центру с эффектом размытия стекла (glassmorphism).
* **Текстовое наполнение**:
  * Заголовок (H2): `Пространство для идеального отдыха`
  * Текст (p): `Просторные дизайнерские номера с премиальной шумоизоляцией и ортопедическими матрасами. Здесь есть всё для глубокого сна после пляжа или продуктивной работы в тишине.`
  * Кнопка (CTA): `Выбрать номер`
* **Фоновое видео**: Кадры интерьера номеров (семейный, полулюкс, стандарт) и детализация постели/комфорта.

### 3. Бассейн и Летняя зона (Pool & Summer Zone) — `#pool-section`
* **Высота**: `100vh`, карточка справа или по центру.
* **Текстовое наполнение**:
  * Заголовок (H2): `Освежающий летний вайб`
  * Текст (p): `Огромный открытый бассейн (80 м²) с лаунж-зоной и удобными шезлонгами. Идеальное место, чтобы поймать ленивое курортное настроение, не покидая территорию отеля.`
* **Фоновое видео**: Кадры бассейна, искрящейся воды, шезлонгов и зоны отдыха.

### 4. Ресторан Redling (Restaurant Redling) — `#restaurant-section`

* **Текстовое наполнение**:

  * Текст (p): `Авторское прочтение черноморской и европейской кухни в стильном классическом интерьере. Комфортная лаундж-зона для неторопливых ужинов и раздельные залы для курящих и некурящих гостей.`

### 5. Коротко об отеле (Atmosphere) — `#atmosphere-section`
